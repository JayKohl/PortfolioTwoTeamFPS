FREE GUNS PACK v1.0

This guns pack contains 4 different guns which can be used in any project. A mobile version of each gun is provided in the mobile folder. Mobile assets have the same tri count but come with an atlased texture. Each gun (excluding the rocket launcher) comes with a reload animation which can be triggered from the animator controller. Enjoy!

- Rocket Launcher: 5439 Tris

- Sci-fi Rifle: 5839 Tris

- Submachine Gun: 6710 Tris

- Shotgun: 4696 Tris
